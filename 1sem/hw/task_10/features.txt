Система учета платежей.
- Консольный интерфейс, принимающий строковые и числовые значения
- Добавление договоров с ID и датой, удаление договоров по их ID
    - Дата неизменяема
- Добавление документов с ID, датой, типом и денежным значением, удаление договоров по их ID
    - Возможность изменять денежное значение документа, дату документа и тип документа
    - Тип документа можно выбрать из двух приводимых (внутреннее реализовано как enum)
    - Денежное значение не может быть меньше или равно 0 - ни при создании договора, ни при его изменении
    - Дата документа не может быть меньше даты родительского договора - ни при создании документа, ни при его изменении
- Возможность получить полный список всех договоров с их датами и их документами,
полностью выводя структуру платежной базы пользователю. Таким образом, выводятся как все платежи и документы,
так и все договоры, давая пользователю всю полноту информации.
Таким образом можно узнать все существующие ID договоров и документов.
- Возможность выводить лишь список договоров либо лишь список документов конкретного договора
- Автоматические подсчет суммы всех документов договоров и демонстрация этих сумм вместе с датами и ID договоров
- Дата проверяется - например, программа выдаст ошибку при вводе 20221320 как даты, т.к. 13 месяца не бывает
- Программа выдает ошибки при неверном вводе и подсказывает, что не так и как правильно вводить значения,
не прекращая работу. Все ошибки из исходного задания обрабатываются.
- ID может быть любой строкой
- Программа на английском, работает с долларами.

- Таким образом, реализовано всё в задании.
